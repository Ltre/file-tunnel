# 即时传输隧道 - 项目概述

## 项目简介

即时传输隧道是一个安全、高效的设备间文件传输和协同编辑平台。通过二维码连接、P2P传输和实时协同编辑，实现了便捷的跨设备数据共享。

## 核心功能

### 1. 设备连接
- **二维码识别**: 通过二维码快速建立设备连接
- **会话管理**: 基于URL hash的会话标识
- **多设备支持**: 支持3台及以上设备同时协作

### 2. 文件传输
- **小文件传输**: ≤5MB文件通过WebSocket即时传输
- **大文件传输**: >5MB文件通过P2P (WebRTC)传输
- **文件预览**: 图片、视频、音频即时预览
- **传输确认**: 大文件需要接收方确认

### 3. 协同编辑
- **实时同步**: 富文本内容实时同步
- **格式支持**: 支持文本格式化、图片插入
- **内容引用**: 可引用已传输文件
- **持久化**: 内容刷新后保持

### 4. 数据存储
- **本地存储**: 所有数据存储在浏览器IndexedDB
- **分布式**: 每台设备独立存储完整数据
- **会话隔离**: 按会话区分数据

## 技术栈

### 前端
- **HTML5/CSS3**: 现代化界面设计
- **JavaScript (ES6+)**: 核心业务逻辑
- **WebRTC**: P2P文件传输
- **IndexedDB**: 本地数据存储
- **Socket.io Client**: 实时通信

### 后端
- **Node.js**: 服务端运行环境
- **Express**: Web框架
- **Socket.io**: 实时通信服务
- **STUN服务器**: NAT穿透

### 安全特性
- **端到端加密**: 传输过程加密
- **CORS控制**: 防止跨域访问
- **速率限制**: 防止滥用
- **会话隔离**: 数据按会话分离

## 部署配置

### 环境要求
- **Node.js**: 14.x 或更高版本
- **操作系统**: Windows/Linux/macOS
- **浏览器**: Chrome 80+, Firefox 75+, Safari 14+

### 部署步骤
1. 克隆或下载项目代码
2. 运行 `npm install` 安装依赖
3. 运行 `npm start` 启动服务器
4. 访问 `http://localhost:3000`

### 端口配置
- **默认端口**: 3000
- **可配置**: 通过环境变量修改
- **防火墙**: 需开放对应端口
