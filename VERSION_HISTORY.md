## 回溯指南

### 1. 查看所有提交历史
```bash
git log --oneline
```

### 2. 查看所有标签
```bash
git tag
```

### 3. 回滚到指定版本

#### 方法一：临时查看旧版本
```bash
# 查看提交哈希
git log --oneline

# 检出指定提交 (临时状态)
git checkout <commit-hash>

# 例如回滚到初始版本
git checkout cf4e5ce
```

#### 方法二：创建新分支基于旧版本
```bash
# 基于指定提交创建新分支
git checkout -b rollback-test <commit-hash>

# 例如
git checkout -b test-initial-version cf4e5ce
```

#### 方法三：硬回滚 (谨慎使用)
```bash
# 永久回滚到指定版本 (会丢失后续所有更改)
git reset --hard <commit-hash>
```

### 4. 比较版本差异
```bash
# 比较两个版本的差异
git diff <commit1>..<commit2>

# 例如比较初始版本和当前版本
git diff cf4e5ce..HEAD
```

### 5. 标签操作
```bash
# 查看特定标签的代码
git checkout v1.0.0

# 删除标签
git tag -d v1.0.0

# 重新打标签
git tag -a v1.0.0 <commit-hash> -m "Initial version"
```
