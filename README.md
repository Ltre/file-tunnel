# 🚀 即时传输隧道

一个安全、快速、去中心化的设备间文件传输和协同编辑工具。

## 功能特性

### 📱 设备连接
- 通过二维码快速建立设备间连接
- 支持多设备同时加入同一个会话
- 基于URL hash的会话标识，无需注册

### 📁 文件传输
- **小文件** (< 5MB): 立即通过Socket.io传输
- **大文件** (> 5MB): P2P WebRTC DataChannel传输，需要接收方确认
- 支持图片、视频、音频直接预览
- 拖拽上传和点击上传两种方式

### 📝 协同编辑
- 实时富文本协同编辑
- 支持插入图片和引用文件
- 发送富文本消息到聊天记录
- 点击预览气泡查看完整内容

### 💾 本地存储
- 所有数据保存到浏览器 IndexedDB
- 分布式存储，每台设备独立保存完整数据
- 按会话区分内容
- 服务器不存储任何用户数据

## 快速开始

### 安装依赖

```bash
cd tunnel
npm install
```

### 启动服务器

```bash
npm start
```

### 访问应用

打开浏览器访问 `http://localhost`

## 使用方法

### 1. 建立连接
1. 在电脑上打开 `http://localhost`
2. 页面会显示二维码和会话ID
3. 用手机扫描二维码，或手动输入带hash的URL
4. 两台设备自动建立连接

### 2. 发送文件
- **拖拽上传**: 将文件拖入上传区域
- **点击上传**: 点击上传区域选择文件
- **小文件**: 立即传输并显示
- **大文件**: 对方需要确认后才通过P2P传输

### 3. 发送文本
- 在输入框输入文字，按回车或点击发送

### 4. 协同编辑
- 在右侧编辑区域输入内容
- 内容实时同步到所有连接的设备
- 使用工具栏格式化文本
- 点击"发送富文本"将内容发送到聊天记录

### 5. 多设备加入
- 任何设备访问相同的hash URL即可加入会话
- 支持3台或更多设备同时协作

## 安全说明

- ✅ 所有文件内容存储在本地浏览器，不上传到服务器
- ✅ 大文件通过P2P传输，不经过服务器
- ✅ 会话ID随机生成，难以猜测
- ✅ 支持接收方确认后才传输大文件
- ⚠️ 会话URL包含hash，分享时请注意安全

## 浏览器兼容性

- Firefox 75+

- 需要支持 WebRTC 和 IndexedDB
